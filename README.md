# Event Management System

This is a frontend-based Event Management System developed using HTML, CSS, and JavaScript.

## Features
- Admin login and dashboard
- Event listing
- Customer registration for events
- Admin can view registered users
- Responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
- GitHub Pages

## Deployment
The project is deployed using GitHub Pages.
